欢乐俄罗斯 Ver 6.1
________________________

本软件是一款对战型俄罗斯方块游戏。游戏采用中文界面，操作简单，极容易上手。
而且对系统配置要求低，在DOS模式就可以运行。并且全面兼容所有版本的Windows
系统。

本游戏采用双人对战模式，支持与电脑对战，可以使用道具，并独创游戏表情，增强
游戏趣味性。

你试过用鼠标玩俄罗斯方块吗？本游戏对鼠标提供支持，使游戏更容易操控。
________________________

游戏前的菜单：
游戏之前的菜单，上、下方向键选择选项；选项中有“<  >”标示的，按左、右方向
键进行选择；没有“<  >”标示的，按Enter或空格进入下一级菜单或进行相应操作；
按Esc键可返回上级菜单。
菜单使用中文界面，通俗易懂，所以详细选项在这里就不详述了。
________________________

游戏界面：
游戏开始后，玩家可以看到屏幕被垂直分成两部分，左边是玩家一，右边是玩家二。
每一边又分成游戏区、下一方块提示区、道具区、表情区等四个区域。

游戏区：游戏开始后游戏区将显示玩家目前游戏情况，玩家可以看到已落下的方块和
正在落下的方块。

下一方块提示区：显示即将落下的下一块方块。

道具区：显示玩家所获得的道具，并显示第一个道具的简要说明。

表情区：模拟玩家的表情。表情根据游戏情况不断变化，或高兴，或悲伤，或恐惧。
________________________

游戏规则：
这是一款对战型俄罗斯方块游戏，玩家依靠自己消层和使用道具来打击对手。当玩家
游戏区域中的砖块累积到顶端时游戏失败。

本游戏可供0～2人同时游戏。可以二人对战，也可以与电脑对战，或者干脆让电脑与
电脑自己对战，你自己就在电脑旁观看。

玩家消层后，将对另一方玩家造成伤害。一次消四层则对对方加三层，已方游戏区
显示“4 Lines!”；一次消三层则对对方加两层，已方游戏区显示“3 Lines!”。
连续消层三次或以上也会使对方加层，已方游戏区显示诸如“3 Combo!”、
“4 Combo!”之类的提示。连续消层的次数越多，对方增加的层数也越多。

玩家可以使用道具对对手进行伤害或者对自己进行救援。具体请参见“道具说明”。
________________________

游戏控制：
为解决键盘的键位冲突问题，本游戏使用了鼠标+键盘的控制方式。玩家可在主菜单
中更改操控设定。在默认状态下，操控设定如下：

                    玩家一              玩家二
方块左移          方向键  左       鼠标左移（不可更改）
方块右移          方向键  右       鼠标右移（不可更改）
快速下落          方向键  下            Num 0
直接下落            空格键             鼠标右键
旋转              方向键  上           鼠标左键
对玩家一使用道具      Z                 Num *
对玩家二使用道具      X                 Num -
切换道具顺序          C                 Num +

注：在单人游戏模式中，玩家一和玩家二的操控设定都对“人脑”一方有效。
例如，在默认的操控设定下，左方向键和鼠标左移都能使己方的方块左移。

此外，在游戏进行中时，Enter为暂停，Esc为返回主菜单。
________________________

道具说明：
玩家进行消层时会在游戏池底部随机产生道具，当玩家将含有道具的层消掉后会得到
这些层所包含的道具。

本游戏提供12种道具，并在游戏中显示道具说明。

这12种道具分别可以：
加一层，加两层，加三层，减一层，减两层，减三层，加速，减速，交换游戏区，
向下塌方，向右推挤，偷窃道具。
